# IT서비스 경쟁 5사 비교 분석 (생성형 AI 활용 미니 프로젝트)

> 생성형 AI를 단순 요약 도구가 아니라 *구조화된 분석가*로 사용해, 국내 IT서비스 경쟁 5사의 전략·투자·ESG·리스크·재무를 동일한 축으로 비교한 소규모 포트폴리오 프로젝트입니다.

## 왜 만들었나

포스코DX는 국내 IT서비스 시장에서 경쟁하지만, 공개된 분석은 대부분 포스코DX 단독 프로파일이거나 업계 1위(삼성SDS)와의 단편 비교에 그칩니다. 매출 체급별로 줄을 세우고 동일 축으로 비교한 프레임이 필요했고, 그 안에서 포스코DX의 2030년 목표(4조원)가 어느 구간에 놓이는지를 *위치*로 읽고 싶었습니다.

부수적이지만 더 중요한 목표는, 경영기획 실무의 핵심 역량인 **"서로 다른 회사를 같은 축 위에 올려 비교 가능하게 만드는 일"**을 작은 산출물로 직접 연습해 보는 것이었습니다.

## 비교 대상 5사 (매출 체급 계단)

포스코DX가 외부 시장에서 실제로 경쟁하는 그룹을 매출 기준으로 계단처럼 배치했습니다.

1. **삼성SDS** — 13.8조원
2. **LG CNS** — 6.0조원
3. **현대오토에버** — 4.3조원
4. **SK AX** (구 SK C&C) — 2.6조원
5. **롯데이노베이트** — 1.1조원

## 핵심 결과

- **소요 시간:** 수동 예상 약 6~8시간 → 실제 **70분** (약 82% 단축)
- **개선 횟수:** 프롬프트 3회 (v1 → v2 → v3)
- **매트릭스:** 5개사 × 12지표 = **60셀**
- **핵심 인사이트:** 포스코DX의 2030 목표(4조원)는 **현대오토에버·SK AX 구간**을 지나는 궤도이며, 이 구간의 공통 플레이북은 *"특화 도메인 × AI"*. 즉 경쟁 레퍼런스를 삼성SDS가 아니라 이 구간으로 재설정해야 한다는 결론.

## 산출물 신뢰도를 만든 프롬프트 규칙 (v3에서 도입)

`prompt_iteration.md`에 전체 기록이 있고, 핵심 3가지는 다음과 같습니다.

1. **숫자는 원문 그대로 인용.** 반올림 금지, 가능하면 출처 문서명·페이지 함께 표기.
2. **단위는 "억원"으로 통일**, 괄호 안에 YoY % 병기.
3. **추정 금지.** 공시가 안 보이면 `공시 확인 필요`로 표기하고 셀을 비워 둘 것.

특히 3번 규칙 덕분에 결과물이 방어 가능해졌습니다. 예를 들어 롯데이노베이트의 영업이익은 공시로 즉시 확인되지 않아 추정값을 채우지 않고 비워 두었습니다.

## 로컬에서 재현하기

```bash
# Python 3.10 이상
pip install openpyxl reportlab

python build_xlsx.py     # → competitor_analysis.xlsx + competitor_analysis_long.csv
python build_pdf.py      # → portfolio.pdf
```

## 기술 메모

- **openpyxl:** 영업이익률 칼럼은 사전 계산값이 아니라 셀 수식으로 작성해, 워크북을 열 때마다 재계산되도록 함.
- **ReportLab:** 내장 CJK 폰트(`HYSMyeongJo-Medium`, `HYGothic-Medium`)를 사용해 외부 폰트 파일 없이 한글 렌더링.
- **Long-format CSV는 의도적인 선택:** 사람이 읽는 비교표는 Excel로, BI 도구가 잘 읽는 형태는 (회사, 축, 값) 한 행씩 풀어 둔 long-format으로 — 같은 데이터를 *용도에 따라 두 형태로 내보내는* 기획 실무 습관을 반영.
